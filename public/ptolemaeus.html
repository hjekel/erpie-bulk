📖 Bibliotheek

Alle inzichten als kaarten. Klik een kaart om de volledige inhoud te zien. Gebruik de zoekbalk om te filteren.

✎ Klik de potlood-knop om een kaart te bewerken.

🗳️ IdeeënBus

Nieuwe inzichten genereren automatisch voorstellen.

  • Goedkeuren → voorstel wordt een TODO actie
  • Afwijzen → voorstel vervalt
  • Later → voorstel blijft staan voor later

Goedgekeurde acties kun je bijhouden met:

  • Status: TODO / IN PROGRESS / DONE
  • Notities: implementatie-aantekeningen
  • Link: GitHub commit of Claude Code sessie URL

👥 Thought Leaders
